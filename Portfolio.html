Meus Projetos

Abaixo estão alguns dos projetos e estudos técnicos que desenvolvi durante a minha graduação em Engenharia de Software:

1. Validador de Documentos (CPF / CNPJ) em Python

Desenvolvimento de um script funcional utilizando a linguagem Python para validar a estrutura e os dígitos verificadores de documentos como CPF e CNPJ. O projeto foi estruturado aplicando lógica de programação pura e manipulação de strings/listas.

2. Modelagem e Diagramação de Sistemas (UML)

Criação e gerenciamento de diagramas técnicos de arquitetura de software (como diagramas de classes, componentes e diagramas de sequência de casos de uso), utilizando ferramentas de modelagem como EdrawMax para documentar e organizar a lógica estrutural antes da codificação.

3. Desenvolvimento deste Portfólio Web

Criação deste web app responsivo construído do zero, utilizando HTML5 estruturado, estilização personalizada com CSS3 (sem frameworks) e lógica de validação de formulários com JavaScript nativo. Hospedado diretamente no GitHub Pages.
